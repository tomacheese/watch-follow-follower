# AI Agents Guidelines

## 目的
このドキュメントは、本リポジトリで作業する全ての AI エージェントが遵守すべき共通の方針とルールを定義します。

## 基本方針
- **会話言語**: 日本語を使用してください。
- **コメント**: コード内のコメントは日本語で記述してください。
- **エラーメッセージ**: 英語で記述してください。
- **コミットメッセージ**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従ってください（`<description>` は日本語）。

## 判断記録のルール
作業において重要な判断を行った場合は、以下の項目を記録・報告してください：
1. 判断した内容
2. 検討した代替案
3. その案を採用した理由
4. 前提条件や不確実性

## 開発手順
1. **プロジェクト理解**: `README.md` やソースコードを読み、目的と構造を把握する。
2. **環境構築**: `pnpm install` を実行する。
3. **実装**: 既存のコーディング規約（ESLint/Prettier 設定）に従い実装する。
4. **検証**: `pnpm lint` および `pnpm lint:tsc` を実行し、エラーがないことを確認する。テストコードは現状ないため、静的解析を重視する。

## セキュリティ / 機密情報
- **コミット禁止**: API キー、パスワード、認証トークンなどは絶対に Git にコミットしないでください。
- **ログ出力**: ログに機密情報が出力されないよう配慮してください。

## リポジトリ固有
- パッケージマネージャーは `pnpm` です。
- ランタイムは Node.js v24.11.1 です。
- TypeScript を使用しています。型定義を適切に行ってください。
